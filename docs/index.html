FragDenStaat App

Wir gehen neue Wege, um den Zugang zur Informationsfreiheit weiter zu vereinfachen: Wir entwickeln eine App. Aktuell kann man mit der App nur IFG-Anfragen lesen und noch keine neuen erstellen. Bis es soweit ist, wird wohl auch noch eine ganze Weile vergehen. Die Funktion kommt aber auf jeden Fall. Die weitere Entwicklung kann man auf GitHub mitverfolgen. Mehr Informationen zu FragDenStaat gibt's in diesem Video.
